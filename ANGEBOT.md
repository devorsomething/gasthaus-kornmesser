## 1. PROJEKTBESCHREIBUNG

Entwicklung einer hochwertigen Website für das Gasthaus Kornmesser in Bregenz. Das Restaurant positioniert sich als gehobene österreichische Gastronomie (€€€) am Kornmarkt.

### Ist-Situation
- Keine eigene Website vorhanden
- Keine domain (gasthaus-kornmesser.at nicht registriert)
- Präsenz nur über Yelp (⭐ 4.2, 21 Bewertungen)
- Telefonnummer nicht öffentlich gelistet

### Zielsetzung
- Professionelle digitale Präsenz
- Online-Reservierungssystem
- Menü-Präsentation mit Preisen
- Kontaktmöglichkeiten (Telefon, E-Mail, Karte)
- Vertrauensaufbau durch Atmosphäre-Bilder

- **Sprache:** TypeScript
- **Styling:** Tailwind CSS v3
- **Hosting:** Static Export (kostengünstig, performant)
- **CDN:** Cloudflare (kostenlos)

### Design-System

- Professionelle Restaurant-Fotos (Küche, Ambiance, Gerichte)
