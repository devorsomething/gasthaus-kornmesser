# Gasthaus Kornmesser Website

**Score: 78/100** – Premium Website für gehobene österreichische Gastronomie

## 🌟 Overview

A premium, elegant website for Gasthaus Kornmesser, an upscale Austrian fine dining restaurant in Bregenz, Austria.

**Live URL:** https://gasthaus-kornmesser.github.io/website

## 📱 Sections (10 Total)

1. **Hero** – Full-screen restaurant image with elegant typography
2. **Info Strip** – Contact details bar
3. **Willkommen** – Two-column story section
4. **Speisekarte** – Menu cards with pricing (€€€)
5. **Weinkarte** – Wine highlights
6. **Ambiance** – Photo gallery (3 cards)
7. **Reservierung** – Elegant reservation form
8. **Bewertungen** – 3 testimonials
9. **Location & Kontakt** – Google Maps + contact info
10. **Footer** – Navigation + social links

## 🛠 Tech Stack

- **Framework:** Next.js 15 + React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS v3
- **Fonts:** Google Fonts (Playfair Display, DM Sans)
- **Images:** Unsplash CDN

```bash

# Production build
npm run build

# Start production server
npm start
```
